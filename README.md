# Должная Душа SWAG 💸💀

Минималистичный и стильный трекер финансов и долгов. Помогает держать под контролем доходы, кредиты, рассрочки и планировать бюджет.

## Возможности

- **Учет доходов:** Фиксированные, плавающие и интервальные поступления.
- **Управление долгами:** Кредиты, кредитные карты, рассрочки, займы и сплиты.
- **Календарь платежей:** Наглядное отображение всех списаний и поступлений.
- **DTI статистика:** Расчет долговой нагрузки (Debt-to-Income) для оценки финансового здоровья.
- **Горизонт планирования:** Подбор оптимального срока закрытия долгов с учетом минимального остатка на жизнь.
- **Экспорт и импорт:** Сохранение данных в JSON.

## Технологии

- React 18+
- TypeScript
- Tailwind CSS
- Vite
- Luxon (для работы с датами)
- Recharts (для графиков)
- Lucide React (иконки)
- Framer Motion (анимации)
